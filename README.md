# 🚚 UrbanLog

**Cada entrega no lugar certo, na hora certa.**

A UrbanLog é uma plataforma de logística urbana desenvolvida para fornecer visibilidade em tempo real das entregas realizadas durante o dia. O sistema permite que gestores acompanhem o andamento das operações sem depender de ligações ou mensagens para os entregadores.

## 🔗 Deploy

**Frontend:** https://urbanlog.onrender.com
**Backend:** https://urban-log.vercel.app

## 📌 Problema

Empresas que realizam múltiplas entregas simultaneamente enfrentam dificuldades para acompanhar o status dos pedidos em tempo real. Muitas vezes, o gestor não sabe quais entregas foram concluídas, quais estão em andamento ou atrasadas, gerando atrasos no atendimento ao cliente e perda de eficiência operacional.

## 💡 Solução

A UrbanLog centraliza todas as entregas em um único painel, permitindo:

- Visualização das entregas em tempo real;
- Controle dos status das entregas;
- Identificação rápida de entregas atrasadas;
- Acompanhamento do entregador responsável;
- Histórico das entregas concluídas.

## 📋 Funcionalidades

- Cadastro de entregas
- Atualização de status
- Listagem de entregas
- Edição de entregas
- Exclusão de entregas
- Painel operacional em tempo real
- Indicadores de desempenho das entregas

Projeto desenvolvido para simular uma solução tecnológica voltada à gestão de logística urbana, proporcionando mais controle, organização e visibilidade das operações de entrega.
